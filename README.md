# FileConverterPro (Python GUI)
A simple python application to convert your files offline.

## 🚀 Features
- **Image Conversion:** JPG, PNG, WEBP, BMP, PDF, TIFF
- **Document Conversion:** Word (.docx) to PDF & PDF to Word (.docx)
- **Modern UI:** Dark mode support with CustomTkinter
- **Multithreading:** Non-blocking background processing
- The converted file is saved to the same location as the original file.
- It doesn't delete the old file.

## 📦 Installation
Import requirements.
Just run the code.
